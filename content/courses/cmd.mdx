## ¿Qué es el simbolo de sistema o command prompt?
El **_simbolo de sistema_** o **_command prompt_** es una herramienta que nos ofrece Windows, 
para poder ejecutar y administrar nuestro sistema operativo. Esta herramienta nos permite
ejecutar comandos y scripts, para realizar tareas de administración del sistema operativo.

Es intérprete de comandos en OS/2 y sistemas basados en Windows NT (incluyendo Windows 2000
y Windows XP). El intérprete de comandos de Windows NT es cmd.exe. El intérprete de comandos
de Windows 9x es command.com, que es una versión reducida del intérprete de comandos de
MS-DOS. El intérprete de comandos de Windows 9x no es compatible con el intérprete de
comandos de Windows NT y de la familia de windows 8.1x

Fue lanzado el año 1987 en el mes de Diciembre y desarrollado en el lenguaje de programación
[C++](https://es.wikipedia.org/wiki/C%2B%2B) por Microsoft y es un tipo de programa [Shell](https://es.wikipedia.org/wiki/Shell_de_comandos).

## ¿Cómo abrir el simbolo de sistema en Windows?
Para abrir el **_simbolo de sistema_** en Windows, debemos seguir los siguientes pasos:

1. Presiona la tecla <kbd>Windows</kbd> + <kbd>R</kbd> para abrir la ventana de ejecutar.
2. Escribe _cmd_ y presiona <kbd>Enter</kbd> o haz clic en el botón **Aceptar**.
3. Se abrirá la ventana del **_simbolo de sistema_**.

Dentro del **_simbolo de sistema_** podemos ejecutar comandos y scripts, para realizar tareas
de administración del sistema operativo, basicamente podemos manejar nuestro sistema operativo
de manera eficaz y rápida.

## Comandos
Un comando es una orden/instrucción es una instrucción que el usuario proporciona 
a un sistema informático, desde la línea de órdenes o desde una llamada de programación. 
Puede ser interno o externo. Suele admitir parámetros o argumentos de entrada, 
lo que permite modificar su comportamiento predeterminado.

### ¿Cómo ejecutar un comando?
Para ejecutar un comando, debemos escribir el nombre del comando y presionar la tecla
<kbd>Enter</kbd> para ejecutarlo dentro del Simbolo de Sistema, es decir si queremos
ejecutar el comando **_dir_** debemos escribir lo siguiente:

```bash title="C:/cmd"
dir
```

### ¿Cómo ejecutar un comando con privilegios de administrador?
Para ejecutar un comando con privilegios de administrador, debemos abrir el **_simbolo de sistema_**
como administrador, para ello debemos seguir los siguientes pasos:

1. Presiona la tecla <kbd>Windows</kbd> + <kbd>R</kbd> para abrir la ventana de ejecutar.
2. Escribe _cmd_ y presiona <kbd>Ctrl</kbd> + <kbd>Shift</kbd> + <kbd>Enter</kbd> o haz clic en el botón **Aceptar**.
3. Se abrirá la ventana del **_simbolo de sistema_** con privilegios de administrador.

De esa forma podremos ejecutar comandos con privilegios de administrador teniendo
el control absoluto de nuestro sistema operativo.

### Comandos básicos de Windows
A continuación te mostraremos los comandos básicos de Windows, para que puedas
utilizarlos en tu día a día o practicar con ellos.

#### <kbd>cd</kbd>
El comando **_cd_** nos permite cambiar de directorio.

```bash title="C:/cmd"
cd C:/Users/leoarayav
```

Asi podemos cambiar de directorio, en este caso cambiamos al directorio **_leoarayav_**.

#### <kbd>dir</kbd>
El comando **_dir_** nos permite listar los archivos y directorios de la ruta actual.

```bash title="C:/cmd"
dir
```

#### <kbd>cls</kbd>
El comando **_cls_** nos permite limpiar la pantalla del **_simbolo de sistema_**.

```bash title="C:/cmd"
cls
```

Si el directorio está vacío no se mostrará nada.

#### <kbd>exit</kbd>
El comando **_exit_** nos permite salir del **_simbolo de sistema_**.

```bash title="C:/cmd"
exit
```

El comando **_help_** nos permite mostrar una lista de comandos disponibles.

El comando **_taskkill_** nos permite finalizar un proceso.


## Argumentos
Un argumento es un valor que se pasa a un programa o a una rutina de subrutina como entrada.
Los programas de línea de comandos, incluidos los scripts de shell, aceptan argumentos de
línea de comandos que se pasan al programa en tiempo de ejecución.

También podemos utilizar argumentos en los comandos, por ejemplo si queremos listar
los archivos y directorios de la ruta actual, pero queremos que nos muestre los archivos
y directorios ocultos, podemos utilizar el argumento **_/a_**.

```bash title="C:/cmd"
dir /a
```

O por ejemplo si queremos listar los archivos y directorios de la ruta actual, pero
queremos que nos muestre los archivos y directorios ocultos y los archivos y directorios
del sistema, podemos utilizar el argumento **_/a:h_**.

```bash title="C:/cmd"
dir /a:h
```

Los argumentos son muy útiles para poder hacer un comando más específico y poder
obtener la información que necesitamos.

Recuerda que la sintaxis es la siguiente:

<pre>comando [argumento]</pre>

## Ventajas de utilizar el simbolo de sistema
- Podemos ejecutar comandos y scripts, para realizar tareas de administración del sistema operativo.
- Podemos manejar nuestro sistema operativo de manera eficaz y rápida.
- Podemos ejecutar comandos con privilegios de administrador teniendo el control absoluto de nuestro sistema operativo.
- Podemos utilizar argumentos en los comandos, para hacerlos más específicos y obtener la información que necesitamos.
- Podemos automatizar tareas con scripts.

Y muchas más ventajas que podrás descubrir a medida que vayas utilizando el **_simbolo de sistema_**.

## Desventajas de utilizar el simbolo de sistema
- No es tan amigable para el usuario.
- No es tan intuitivo.
- No es tan visual.
- No es tan fácil de utilizar.
- No es tan fácil de aprender.

Por eso es super importante que aprendas lo básico, para que puedas utilizarlo
y que a la hora de administrar tu sistema operativo o ejecutar algun comando,
tengas alguna base de conocimiento mínima para poder hacerlo.
